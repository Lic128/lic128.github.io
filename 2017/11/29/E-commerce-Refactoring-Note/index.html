E-commerce Refactoring Note

Posted on 2017-11-29

Q: Project Introduce

Technique

The whole application is built on Node.js + Express+ MongoDB+ Bootstrap+Passport;

The backend is based on Node.js and Express. We use mongoose to connect the database. There are three schemas in the application: User, Item and Comment, all the operation in the application is actually operation on the three tables.

while in the mean time, passport was used to perform authentication. A user can register, login, and logout. According to the different status user in, he can perform different operations.

And as you can see here, the front end is based on Bootstrap, and in the next version, I may use vue.js or react to update the frontend.

what have I learned from the process.

Through the building process, I have a deeper understanding of how the node.js works, and how it cooperate with the frontend. In the meantime, I’m also more familiar with the frontend. By searching on website, I have solved different problems such as the different strategy the passport should use, and the how to design and implement database. In a word, I’m more confident to build to more complicate web application.
